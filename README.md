# CSS-Training-Course

***What is CSS?***
  + **cascading Stylesheets**
  + **NOT** a programming language
  + Stylesheet/Styeling language
  + Used for website layout and design 
  + Can be extended with Sass/Less

    - Opera 

  + A Test Editor (Pink One)
    + Visual Studio Code
    + Sublime Text 
    + Atom.io
    + Notepad++ (Windows)
    + TextMate (Mac)
